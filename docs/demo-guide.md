# 海南国际清算所报价交易平台 - 演示总纲 (Demo Master Guide)

本手册旨在指导演示人员通过“演示导向”的方式测试平台功能并进行竞标演练。

## 1. 演示准备 (Setup)

### 1.1 启动环境
1. 确保在终端运行 `npm run dev`（本地测试）或直接访问在线地址。
2. 在浏览器打开 `https://hni-trade-demo.vercel.app/`。

### 1.2 准备演示窗口 (建议：双窗口并排)
为了体现“实时通讯”和“撮合效率”，强烈建议采用**双窗口联动**：
*   **窗口 A**: 打开一个普通浏览器标签，访问 `https://hni-trade-demo.vercel.app/`。
*   **窗口 B**: 打开一个**无痕/私密窗口**（或另一个浏览器，如 Chrome vs Edge），访问 `https://hni-trade-demo.vercel.app/`。
    *   *注：本演示系统支持多实例实时同步。*

### 1.3 身份登录
*   在 **窗口 A** 登录选择：`海汽大宗 (买方)`。
*   在 **窗口 B** 登录选择：`沙钢贸易 (卖方)`。
*   (可选) 如需演示人工干预，可在 **窗口 C** 登录：`HNI 监管端 (管理员)`。

## 3. 测试与反馈建议
*   **数据一致性**: 在窗口 A 所有的变动，必须在 0.5 秒内同步到窗口 B。
*   **UI 响应**: 报价成功后应有明显的视觉反馈。
*   **身份感**: 确保顶栏显示的机构名称与当前操作逻辑完全吻合。
