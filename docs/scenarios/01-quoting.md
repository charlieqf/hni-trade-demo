# 演示脚本 01: 买卖报价与市场深度 (基差定价模式)

## 场景目标
演示专业的大宗商品报价流程，包括：
1. **基差定价模式**：如何关联期货参考价进行快速升贴水报价。
2. **前置风控演练**：在发布报价前，系统如何实时测算预估保证金。
3. **市场实时深度**：不同定价模式下的报价如何统一在行情版块展现。

## 角色分配
*   **角色 A**: 沙钢贸易 (卖方) - 信用评级: **AAA**
*   **角色 B**: 海汽大宗 (买方) - 信用评级: **AAA**

## 演示准备
1. **启动**: 请确保已开启两个独立窗口并登录对应身份。
2. **定位**: 双方均选择 `钢材` -> `螺纹钢` 作业区。

## 演示流程

### 步骤 1: 卖方发布基差报价 (专业级演示)
1.  **操作**: 角色 A (卖方) 点击报价页上方的 `基差报价 (PRO)` 标签。
2.  **动作**: 
    *   观察系统自动抓取的期货主力报价（如 RB2505: 3820）。
    *   在“升贴水”处通过 `+` 按钮调整至 `+30`。
    *   填入品种属性：`品牌: 沙钢`, `规格: Φ18-25`。
3.  **风控演示**: 观察报价按钮上方的**结算风控预估**版块。随着数量（设为 200 吨）的调整，预估缴纳保证金会实时变动。
4.  **执行**: 点击 `发布卖出报价`。
5.  **解说**: *“在专业大宗交易中，基差定价是标准。我们的系统允许交易员直接对标期货主力合约进行加价，同时利用分布式清算引擎在‘事前’就完成保证金的压力测算，确保交易链条的安全。”*

### 步骤 2: 买方响应并建立反向买盘
1.  **操作**: 窗口 B (买方) 看到卖方挂出的 `3850` 吨价，决定报入买盘。
2.  **方式**: 角色 B 可以选择 `普通价报价`，手动输入其心理价位（如 `3830`）。
3.  **结果**: 点击发布。观察中间行情板，买盘 `3830` 与卖盘 `3850` 形成买卖点差。
4.  **解说**: *“买方选择了普通定价。系统兼容多种定价范式，并在统一的市场深度中进行全量撮合。”*

### 步骤 3: 资产端管理
1.  **操作**: 双方点击顶部的 `我的资产`。
2.  **视觉重点**: 
    *   观察机构名称旁的 **AAA 信用授信勋章**，这展示了清算所对入场机构的动态准入评估。
    *   在“当前挂单”中管理刚才发布的订单。

## 测试验证点
- [ ] **信用评级**：导航栏是否显示了 AAA/AA 的专业评级勋章？
- [ ] **基差联动**：修改升贴水后，系统底部的“折算现货报价”是否自动联动？
- [ ] **风控测算**：在发布报价前，是否看到了显眼的“预估保证金”数值？
- [ ] **撤单同步**：在资产页点击撤单后，中间行情板对应的挂单是否实时移除？
