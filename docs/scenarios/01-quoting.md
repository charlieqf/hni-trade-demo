# 演示脚本 01: 买卖报价与市场深度

## 场景目标
演示不同参与者（买方/卖方）如何发布报价，以及系统行情版块（五档行情）如何实时反映市场动态。

## 角色分配
*   **角色 A**: 沙钢贸易 (卖方)
*   **角色 B**: 海汽大宗 (买方)

## 演示准备
1. **启动**: 参考[演示总纲](file:///c:/work/code/hni_trade/docs/demo-guide.md)并开启两个窗口。
2. **窗口 A (卖家)**: 登录 `沙钢贸易`，进入系统。
3. **窗口 B (买家)**: 登录 `海汽大宗`，进入系统。
4. **同步**: 确保两个窗口均在左侧边栏选择了 `钢材` -> `螺纹钢`。

## 演示脚本

### 步骤 1: 卖方发布初始价格
1.  **操作**: 角色 A 进入系统，选择 `螺纹钢`。
2.  **动作**: 在右侧报价单，输入 `品牌: 沙钢`, `规格: Φ18-25L`, `价格: 3950`, `数量: 500`。点击 `发布卖出报价`。
3.  **视觉重点**: 观察中间区域的五档行情。`卖1` 处出现 `3950` 的挂单，数量 `500`。
4.  **解说**: *“卖方基于实盘货源发布报价，该信息已实时广播至全场，所有人都能在行情版块看到挂单。”*

### 步骤 3: 挂单管理与验证
1.  **操作**: 点击顶部导航 `我的资产`。
2.  **视觉重点**: 
    *   在“当前挂单”列表中，能看到角色 B 刚刚挂出的 `3880` 螺纹钢买单。
    *   点击最右侧的 `X` 按钮尝试撤单。
3.  **结果**: 观察中间区域，“买1”挂单实时消失。
4.  **解说**: *“交易员可以随时在资产管理页管理自己的意向单，撤单动作通过底层协议即时同步至全网行情面板。”*

### 步骤 3: 挂单管理与验证
1.  **操作**: 点击顶部标签 `我的资产`。
2.  **视觉重点**: 
    *   在“当前挂单”列表中，能看到角色 B 刚刚挂出的 `3880` 螺纹钢买单。
    *   点击该订单最右侧的 `X` 按钮执行撤单。
3.  **结果**: 观察中间行情区域，“买1”价位的挂单实时消失。
4.  **解说**: *“交易员可以随时在资产管理页查看并撤回自己的报价，挂单状态的变更会通过分布式总线即时同步至全网行情面板。”*

## 测试验证点
- [ ] 卖方挂单后，买方界面是否即时刷新显示？
- [ ] 修改品种（如从钢材切换到铁矿）后，报价表单的属性字段是否正确变换？
- [ ] 撤单操作：点击 `我的资产` 中的撤单，行情版块对应的挂单是否消失？
- [ ] 挂单成功后，点击“我的资产”是否能实时看到该笔“当前挂单”？
- [ ] 发布成功后，点击“我的资产”是否能看到对应的“当前挂单”？
