# 演示脚本 02: 做市商双向报价 (流动性提供)

## 场景目标
演示专业做市商如何利用“快速双边报价”功能，同时在买卖两端提供流动性，并收窄市场买卖价差。

## 角色分配
*   **角色 A**: 宏源做市 (做市商)
*   **角色 B**: 全场观察者 (任意角色)

## 演示准备
1. **窗口 A (做市商)**: 登录 `宏源做市 (做市商)`。
2. **窗口 B (观察者)**: 登录 `海汽大宗` 或 `沙钢贸易` 用于观察行情变化。
3. **数据初始状态**: 确保 `螺纹钢` 市场已有少量零散挂单，以便演示做市商如何收窄价差。

## 演示脚本

### 步骤 1: 做市商进入并发布流动性
1.  **操作**: 角色 A 进入系统，选择 `螺纹钢`。使用 `快速双边报价` 面板。
2.  **动作**: 观察当前市场最优买价为 `3880`，最优卖价为 `3950`。做市商为了活跃市场，输入 `价格: 3910`，点击发布。
3.  **视觉重点**: 
    *   报价单右侧显示系统自动推算的双向报价：`买入 3905 / 卖出 3915` (假设价差设为 ±5)。
    *   点击 `发布双边快速报价`。
4.  **行情变化**: 观察中间五档行情。`买1` 立即变为 `3905`，`卖1` 变为 `3915`。买卖价差显著收窄。
5.  **解说**: *“做市商通过‘双边报价’同时报出买入和卖出价，有效降低了市场的交易成本，为买卖双方提供了即时成交的机会。”*

## 测试验证点
- [ ] 双边报价是否能够一键同时生成两笔挂单？
- [ ] 做市商发布的买价是否排在普通买家之上（如果价格更高）？
- [ ] 做市商看板左下角的“做市商公告”是否显示了合理的提示文本？
